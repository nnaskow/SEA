🏢 Syndiceo – Интегрирана система за управление на етажна собственост
Syndiceo е мащабна софтуерна платформа, изградена върху .NET 8, предназначена за дигитализация на административните и финансовите процеси в многофамилните жилищни сгради. Проектът използва хибриден подход, съчетавайки мощно десктоп приложение за домоуправителите и интерактивна уеб платформа за живущите.

🛠 Технически архитектурен стек
Core Framework: .NET 8 (LTS)

Database: Microsoft SQL Server

ORM: Entity Framework Core (с Fluent API конфигурация)

Identity Management: ASP.NET Core Identity (интегрирано в SyndiceoDBContext)

Architecture: Layered Architecture с използване на Partial Classes и Fluent API за прецизно дефиниране на релациите.

📊 Структура на базата данни (Entity Models)
Системата управлява сложна релационна схема, включваща следните ключови модули:

🏠 Управление на собствеността
Addresses, Blocks, Entrances: Йерархична структура на местоположението.

Apartments: Централен обект, свързващ живущите с физическите имоти.

Owners: Детайлна информация за собствениците и техните контакти.

💰 Финансов модул (Сърцето на системата)
ApartmentTransactions: Проследяване на личните вноски на всеки апартамент.

EntranceTransactions: Регистър на разходите на целия вход (ток, асансьор, чистачка).

Cashbox: Автоматизирано следене на текущия баланс (касова наличност) за всеки вход.

Debts: Динамично изчисляване на задълженията чрез Computed Column Sql за автоматично генериране на остатъчни суми.

Categories: Класификация на транзакциите за по-добра отчетност.

📢 Комуникация и поддръжка
Discussions & DiscussionReplies: Пълнофункционален модул за вътрешни форуми с поддръжка на нишки от отговори.

Reports: Система за сигнализиране за повреди с проследяване на статус (IsResolved).

Maintenance: История на извършените ремонти и техническа поддръжка.

Documents: Модул за съхранение и управление на цифрови копия на договори и фактури.

🔑 Ключови функционалности в кода
Identity Integration: Контекстът наследява IdentityDbContext<SyndiceoWebUser>, което позволява безпроблемна връзка между системните потребители и техните апартаменти.

Fluent API Configuration: Всяка релация е прецизно дефинирана с правила за каскадно изтриване (DeleteBehavior.Cascade) или запазване на данни при изтриване на родител.

Computed Columns: Автоматично изчисляване на RemainingSum в таблица Debts директно от SQL сървъра: ([TotalSum]-[PaidSum]).

Multi-Active Result Sets (MARS): Активирано за подобряване на производителността при сложни заявки към базата данни.

🚀 Инсталация и настройка
Конфигурация на Connection String:
Проектът е настроен да работи с отдалечен SQL сървър. За локална разработка, променете стринга в OnConfiguring:

C#

optionsBuilder.UseSqlServer("Server=YOUR_SERVER; Database=SyndiceoDB; Integrated Security=True;");

Миграции:
Използвайте следните команди в Package Manager Console:

PowerShell

Add-Migration InitialCreate

Update-Database

Стартиране:

Syndiceo.Web – Стартирайте за достъп до портала за живущи.

Syndiceo.Desktop – Стартирайте за административния панел на домоуправителя.

📂 Проектна структура
Syndiceo.Data.Models – Съдържа SyndiceoDBContext и всички POCO класове.

SyndiceoWeb.Areas.Identity – Управление на потребителските профили и сигурността.
