🏢 Syndiceo – Професионален домоуправител
Syndiceo е хибридна софтуерна екосистема, създадена да дигитализира и автоматизира управлението на етажната собственост. Системата разделя административните задачи (десктоп) от нуждите на живущите за информация и комуникация (уеб).

🚀 Основни функционалности
💻 Десктоп приложение (За администратори)
Управление на обекти: Пълно CRUD управление на адреси, блокове и входове.

Финансов контрол: Автоматично начисляване на месечни такси и фонд "Ремонт".

Отчетност: Генериране на детайлни месечни отчети в Word формат за всеки апартамент и общ отчет за входа.

Архивиране: Автоматично и ръчно създаване на архиви (Backup) и възстановяване на базата данни.

🌐 Уеб платформа (За живущи)
Личен профил: Проверка на текущи задължения и история на плащанията в реално време.

Сигнали: Подаване на сигнали за технически проблеми (изгорели крушки, повреди) с проследяване на статуса.

Дискусии: Участие в отворени чатове за обсъждане на въпроси, касаещи входа.

Сигурност: Двуфакторна верификация – потвърждение по имейл и ръчно одобрение от домоуправител за конкретен адрес.

🛠 Технологичен стек
Framework: .NET 8 (LTS)

Език: C#

База данни: MS SQL Server

ORM: Entity Framework Core

Security: ASP.NET Core Identity (паролно хеширане и роли)

Уеб Frontend: ASP.NET Core MVC, Bootstrap 5, Razor Pages

Десктоп Frontend: WPF

Hosting: MonsterASP.net (за уеб частта)

📂 Структура на проекта
Syndiceo.Web – Уеб портал за живущите.

Syndiceo.Desktop – Приложение за домоуправителя.

Syndiceo.Data – Споделен слой за достъп до данни (Models, Context).
